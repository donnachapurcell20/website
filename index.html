Projects

Destination A Route Planner App

A route planner app that was creating using Java in Android Studio IDE. This app allows users to input the starting and end locations, provide route suggestions, display speed van and speed camera locations across Ireland, and offer direction. users could activiely contribute to the app by adding markers, provind valuable input to improve the user experience. These markers, categorised as "accident", "construction", "speed van", or "speed camera", which were seamlessly integrated into the apps and stored on Firebase Cloud, further enhancing its functionality.
